# Sanjarbek Otabekov - Professional Portfolio

Zamonaviy, interaktiv dizaynga ega hamda yuqori tezlikda ishlovchi professional portfolio veb-ilovasi. Ushbu loyiha mening IT sohasidagi tajribamni, ko'nikmalarimni va amalga oshirgan loyihalarimni namoyish etish uchun maxsus ishlab chiqilgan.

## 🚀 Texnologiyalar

Loyiha bugungi kunning qulay, tezkor va ishonchli veb-texnologiyalari asosida qurilgan:

- **Platforma (Framework):** [Next.js 15](https://nextjs.org/) (App Router interfeysi)
- **Dasturlash tili:** [TypeScript](https://www.typescriptlang.org/)
- **Dizayn va Stilizatsiya:** [Tailwind CSS v4](https://tailwindcss.com/)
- **Animatsiyalar:** [Framer Motion](https://www.framer.com/motion/) (Motion)
- **Ikonkalar:** [Lucide React](https://lucide.dev/)

## 🛠 O'rnatish va Ishga tushirish

Loyihani kompyuteringizda (mahalliy muhitda) ishga tushirish uchun quyidagi ko'rsatmalarni bajaring:

1. **Loyihani yuklab oling (Clone):**
   ```bash
   git clone https://github.com/sanjarbek404/NexT-JS-Portfolio.git
   cd <portfolio>
   ```

2. **Dasturiy qaramliklarni (dependencies) o'rnating:**
   ```bash
   npm install
   # yoki
   yarn install
   ```

3. **Dasturni dasturchu muhitida (development) ishga tushiring:**
   ```bash
   npm run dev
   # yoki
   yarn dev
   ```

4. Dasturni ko'rish uchun manzillar qatoriga quyidagi havolani kiriting: `http://localhost:3000`

## 📁 Loyiha Tuzilishi

Loyiha toza arxitektura qoidalari asosida tashkil etilgan:

- `/app` - Next.js marshrut (route) va layout fayllari
- `/components` - Qayta ishlatiluvchi React UI komponentlari
  - `/sections` - Asosiy sahifa bo'limlari (Hero, About, Projects, Skills, Contact)
  - `/shared` - Umumiy qo'llaniladigan qismlar (Navbar, Footer kabi)
- `/public` - Statik fayllar (rasmlar va boshqalar)
